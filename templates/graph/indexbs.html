Help

  • Select an annotation from the list and state which row holds the column names using 'Header'.
  • If the annotation is an Excel file you will also get to choose which sheet contains the data to be plotted.
  • Set 'Header' to 0 (zero) if there are no column names.
  • You can get a preview of what the data (stored in the annotation) looks like using the 'Preview' tab. This can help you decide which columns to plot and which row holds the column names (if any).
  • The graph is plotted using Plot.ly. If you click 'Edit chart' on the bottom left of the chart area you will be taken to the Plot.ly website where you will be able to edit, save and share the chart.
